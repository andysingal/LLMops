[super_proxy](https://github.com/Nextbasedev/super-proxy)

- Routes across Anthropic, OpenAI, Groq, Cerebras, Kimi, GLM, Gemini, OpenRouter, xAI, plus Deepgram, Fish Audio, Runpod

- OpenAI-compatible chat/completions/responses routes and Anthropic /v1/messages, streaming and non-streaming

- Provider pools with fallback, so a request can move to another provider when one is rate-limited or down

- Gateway tokens with per-token model and spend limits, revocable from a dashboard, with usage and cost tracking per token

- SQLite persistence, Docker Compose deploy, Apache-2.0
